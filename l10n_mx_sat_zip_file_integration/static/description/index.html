Lectura de XML

Este modulo complementa la Descarga del SAT, permitiendo a los usuarios leer los documentos descargados y en dado caso insertarlos en una Poliza o generar una Factura.

Este modulo agrega una nueva funcionalidad con el boton Leer y Auditar Zip, lo que hace es utilizar el ZIP descargado por el SAT leer sus atributos relacionarlo en la pestaña correspondiente:

  • Ingresos
  • Egresos
  • Traslados
  • Nomina
  • Pago
  • Otros
