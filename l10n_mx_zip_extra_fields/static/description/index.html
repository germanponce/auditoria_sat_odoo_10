Campos Extra para el Auditor CFDI

Este modulo agrega y extrae de los XML procesados en el Auditor de CFDI los siguientes valores:


  • ★   Folio
  • ★   Serie
  • ★   Uso de CFDI
  • ★   Forma de Pago
  • ★   Moneda
  • ★   Tipo de Cambio
  • ★   Sello SAT
  • ★   Sello Emisor
  • ★   Calcula la Cadena Original
  • ★   Genera su codigo Bidimensional
  • ★   UUID
  • ★   Subtotal
  • ★   Total
  • ★   Datos del Emisor
  • ★   Datos del Receptor
  • ★   Conceptos del XML
  • ★   Impuestos
